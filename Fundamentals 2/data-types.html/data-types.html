Basic Data Types

"Strings" : a series of characters within double quotes " " or single quotes '' . Strings cannot be changed only replaced.
Numbers : with or without decimals. If you use a number within a double or a single quote, it will be treated as a string and not as a number.
BigInit : an integer with arbitrary precision.
Booleans have two values : true & false.
Undefined : a variable that a value hasn't been assigned.
Null : has only one value, null, or as in Python (that makes more sense) None.
