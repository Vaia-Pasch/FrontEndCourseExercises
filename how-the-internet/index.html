Brief introduction to the Internet

The internet is a worldwide network that connects computers all over the world. Simply put, internet is a wire.

How does the internet work?

Internet moves data from one computer to another, when we browse and we request a site, when we chat or share information. It uses packets that follow Internet Protocol (IP) and Transport Control Protocol (TCP).

Internet Protocols

Internet protocols are rules in order to govern how devices should communicate one another.

IP Addresses

IP address is a numerical label that has a unique recipient. That way your device can be identified among billions of others.

DNS

DNS stands for Domain Name System. Since we can't remember rows of numbers, DNS eliminates that need by providing domain names.

Packets(TCP/UDP)

Packets are blocks of data, filled with pieces that the minute they reach their destination they reassemble. UDP allows communication between the client and the server. TCP makes sure that the packets arrive to the destination successfully and with the correct order.

HTTP

HTTP stands for HyperText Transfer Protocol and it controls how the page is transferred from the server to the client.

HTML

HTML is a HyperText Markup Language and is our structure. It defines all the content, the images and the links. HTML is on almost every single page on the internet.

Routers

Routers are networking devices. When a packet comes in on one of the lines, the router reads the network address information in the packet header to determine the ultimate destination.
